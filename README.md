# Smart Supply Chain Management System 🚚

## 📌 Overview
This project is a Smart Supply Chain Management System designed to improve logistics efficiency, order tracking, and inventory management.

The system allows users to place orders, administrators to manage them, and provides real-time status updates for delivery tracking.

---

## 🎯 Features
- User Login (Basic Authentication)
- Product Listing
- Order Placement
- Admin Dashboard
- Order Status Tracking (Pending → Shipped → Delivered)

---

## 🛠️ Tech Stack
- Backend: Node.js, Express.js
- Frontend: (Planned for future implementation)
- Data: In-memory (prototype version)

---

## 🔄 Workflow
1. User logs in  
2. Places an order  
3. Admin processes the order  
4. Order status is updated  
5. User tracks delivery
